# SmartLibrary Database Project

## Tables Used

1. Authors
   Stores information about book authors.

2. Books
   Stores book details such as title, category, price, available copies, and author reference.

3. Members
   Stores member details including membership date and contact information.

4. Transactions
   Stores borrowing and returning details of books along with fine amount.

---

## Key Features Implemented

1. CRUD Operations
   Insert, update, delete and retrieve data.

2. Filtering & Sorting
   Using WHERE, HAVING, ORDER BY and LIMIT.

3. Logical Operators
   Using AND, OR and NOT conditions.

4. Grouping & Aggregation
   Using GROUP BY, COUNT, SUM, AVG functions.

5. Primary and Foreign Keys
   Maintains relationships between tables.

6. Joins
   Inner Join
   Left Join
   Right Join
   Full Outer Join using UNION

7. Subqueries
   Nested queries for advanced filtering.

8. Date Functions
   YEAR, DATEDIFF, DATE_FORMAT.

9. String Functions
   UPPER, TRIM, IFNULL.

10. Window Functions
    RANK
    Cumulative Count
    Moving Average

11. Case Expressions
    Used to categorize membership status and books based on conditions.

---

After completing this project, you can:

* Design relational databases
* Create tables with proper keys
* Perform CRUD operations
* Write complex queries
* Use joins and subqueries
* Apply aggregate and window functions
* Use CASE expressions for conditional logic
